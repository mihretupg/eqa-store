# Eqa Storefront

Eqa is a modern ecommerce storefront built with React, TypeScript, Vite, and Tailwind. It showcases a clean shopping experience with product discovery, deals, cart and checkout flows, and polished motion throughout the UI.

## Features
- Multi-page storefront with React Router (Home, Products, Deals, Categories, Cart, Checkout, Orders, Account)
- Product list and detail views
- Cart and checkout experience
- Promotional hero sections and animated highlights
- Responsive layout for desktop and mobile

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Router

## Getting Started
```bash
npm install
npm run dev
```

## Scripts
- `npm run dev`: Start local dev server
- `npm run build`: Type-check and build
- `npm run lint`: Lint the project
- `npm run preview`: Preview the production build
